Resource page for

  • All programs from the book. On Unix/Linux or Mac, download the tarfile for the 3rd edition and pack it out by 
    tar xvzf book-examples-3rd.tar.gz
    On Windows, download the zipfile for the 3rd edition and pack it out by 
    unzip book-examples-3rd.zip
    The result is a folder (directory) tree src with subfolders containing the program examples for different chapters in the book. The introduction to each chapter tells the name of the corresponding subfolder in the src tree.

  • Software installation. There is a brief description of how to install the software needed for working with the book. More updated, comprehensive instructions have been written for the upcoming 4th edition of the book. You may find this latter description more useful.

  • On Debian systems, including Ubuntu, SciTools can be installed by 
    sudo apt-get install python-scitools
    However, it is often advantageous to install the latest version of SciTools from googlecode.com/p/scitools. This is anyway the recommended approach on non-Debian Linux systems, Mac, and Windows. Make sure you have the Mercurial (hg) version control system on your computer (if not, do sudo apt-get install mercurial on Debian/Ubuntu or google for "Mercurial" and follow the installation instructions on Mac and Windows machine). You can download the SciTools source code and install it by running the following commands in a terminal window (Linux/Mac) or in a DOS Window or PowerShell window on Windows: 
    hg clone https://langtangen@code.google.com/p/scitools/
cd scitools
sudo python setup.py install
    The advantage of this installation method is that you can get SciTools updates (bugfixes, enhancements) whenever you want by going to the scitools folder and running 
    hg pull
hg update
sudo python setup.py install

  • You may need additional software. On Debian/Ubuntu it is recommended to do 
    sudo apt-get install python-matplotlib python-scipy python-gnuplot gnuplot gnuplot-x11
    A comprehensive installation with lots of useful packages can be performed by a script install_rich.sh that is run by bash install_rich.sh on a Debian/Ubuntu machine.

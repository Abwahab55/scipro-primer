scipro-primer by hplgit

Resources for the book "A Primer on Scientific Programming with Python"

  • Errata list for the 3rd edition of the book

  • All programs from the book: tarfile for the 3rd edition (pack out: tar xvzf book-examples.tar.gz), zipfile for the 3rd edition (pack out: unzip book-examples.zip),

  • Get the latest version of SciTools from googlecode.com/p/scitools (install by just running sudo python setup.py install).

  • Slides from lectures based on the book.

  • How to install the software needed for working with the book.

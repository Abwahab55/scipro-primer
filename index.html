Resource page for

  • All programs from the book. On Unix/Linux or Mac, download the tarfile for the 3rd edition and pack it out by 
    tar xvzf book-examples.tar.gz
    On Windows, download the zipfile for the 3rd edition and pack it out by 
    unzip book-examples.zip
    The result is a folder (directory) tree src with subfolders containing the program examples for different chapters in the book. The introduction to each chapter tells the name of the corresponding subfolder in the src tree.

  • Software installation. We have made a description of how to install the software needed for working with the book.

  • On Debian systems, including Ubuntu, SciTools can be installed by 
    sudo apt-get install python-scitools
    However, it is often advantageous to install the latest version of SciTools from googlecode.com/p/scitools:

    • Method 1: download tarfile of the latest release from http://code.google.com/p/scitools/downloads/list. 
      tar xzf scitools-0.9.0.tar.gz
cd scitools-0.9.0
sudo python setup.py install
      (the version number, here 0.9.0, might be different)

    • Method 2: install the most recent version 
      hg clone https://langtangen@code.google.com/p/scitools/
cd scitools
sudo python setup.py install
      The advantage of method 2 is that you can get SciTools updates (bugfixes, enhancements) whenever you want by going to the scitools folder and running 
      hg pull
hg update
sudo python setup.py install
